### 1 代谢导论回顾

- 定义
  - Catabolism 分解代谢：营养物/细胞组分降解，回收、供能。
  - Anabolism 合成代谢：由简单组分生物合成复杂分子。
- 三阶段框架
  - 阶段1：大分子↔单体（多糖/脂质↔单糖/脂肪酸/甘油等）
  - 阶段2：单体↔更简单有机物
  - 阶段3：与无机物互转（CO2/H2O/NH3）
- 代谢途径特征
  - 连续性（产物即下一步底物）；多步离散；可调控；可进化。
- 区室化（Compartmentation）
  - 真核细胞代谢在特定细胞器中进行，需跨膜转运。
  - 意义：隔离竞争反应、优化微环境、提高局部浓度、储存产物、实现级联。
  - 应用：人工细胞器/蛋白质胶囊/液-液相分离/细菌微室等，促进合成、隔离旁路、提高产率。
- 课前小测：为什么葡萄糖完全氧化分解为ATP有32 与30分子ATP的区别（两种电子传递方式消耗的ATP不同）

### 2 热力学与氧化还原基础

- 吉布斯自由能
  - ΔG = ΔH − TΔS；ΔG = ΔG′° + RT ln(Q)
  - ΔG < 0 反应自发向正向；ΔG = 0 平衡；ΔG > 0 倾向逆向。
  - 标准生化条件：pH 7，25°C，1 atm，活度~1 M。
  - 关联：ΔG′° = −RT ln Keq
- “高能化合物”
  - 定义：水解ΔG′° < −25 kJ/mol（数值更负）。
  - 代表：PEP（磷酸烯醇式丙酮酸）、1,3-BPG（1,3-二磷酸甘油酸）、磷酸肌酸等。
  - 磷酸基转移电位排序驱动磷酸基自高能向低能流动（激酶催化）。
- 氧化还原与能量
  - Nernst 方程；还原电位 E′° 越正，受电子能力越强。
  - 关系式：ΔG′° = −nFΔE′°（n为转移电子数，F为法拉第常数）。

### 3 电子载体与还原当量

- 电子转移方式
  - 直接电子转移；氢原子转移（H·）；氢负离子（:H−）；与氧气结合。
  - **还原当量**（reducing equivalent）：常以NADH、FADH2、QH2或H−计。
- 通用载体
  - NAD+/NADH、NADP+/NADPH、FAD/FADH2。
  - NAD+ 接受氢负离子（:H−）还原为 NADH；烟酰胺环可逆还原。
- 代谢分工与池浓度
  - [NAD+]/[NADH] 高，利于氧化（分解代谢）；[NADPH]/[NADP+] 高，利于还原（合成代谢）。
  - NAD+ 多参与氧化/产能；NADPH 供还原力（脂肪酸/胆固醇合成、抗氧化等）。
- 氧化还原酶、脱氢酶

### 5 糖的种类、消化吸收与细胞摄取

- 分类
  - 单糖：葡萄糖、果糖、半乳糖、核糖、2-脱氧核糖等；醛糖/酮糖；D-对映体为主。
  - 二糖：麦芽糖、蔗糖、乳糖；多糖：淀粉、糖原、纤维素。
- 消化与酶
  - 淀粉/糖原：α-淀粉酶切α-1,4键；麦芽糖/三糖：麦芽糖酶、α-葡萄糖苷酶；
    蔗糖：蔗糖酶（invertase）；乳糖：乳糖酶。
- 小肠吸收产物
  - 主要为葡萄糖/果糖/半乳糖，经门静脉入肝。
- 细胞摄取（GLUT家族）
  - GLUT1：广泛；GLUT2：肝/胰β细胞/肠；GLUT3：神经元；GLUT4：骨骼肌/心肌/脂肪，受胰岛素刺激转位；
    GLUT5：果糖转运。
  - 胰岛素激活受体酪氨酸激酶→GLUT4膜转位，促进摄糖。

### 6 糖酵解（Glycolysis）总览与发现史

- 定义与特征
- - 一分子葡萄糖转化为两分子丙酮酸，分为有氧和无氧。
  - 无需O2，胞质内进行；10步酶促，所有中间体均为磷酸化化合物。
  - 中心代谢通路，**进化上高度保守**；第一条被阐明的代谢途径之一。
- 发现史要点
  - Pasteur：发酵与酵母细胞相关；
  - Buchner：无细胞提取物可发酵（生化学诞生）；
  - Harden & Young：无细胞中的反应可以通过无机磷促进；中间体为磷酸化糖 果糖1,6-二磷酸 (Fructose 1,6-bisphosphate)；
  - Embden–Meyerhof–Parnas（EMP）途径建立。
- 葡萄糖的四大命运
  - 细胞外多糖（ECM/细胞壁）、储存（糖原/淀粉/蔗糖）、糖酵解供ATP/中间体、磷酸戊糖途径产R5P与NADPH。
- 重要生理
  - 脑（非饥饿）与红细胞主要依赖葡萄糖；RBC完全依赖糖酵解供能。
- 总览
  **十步反应 六种反应类型 所有的酶来自细胞质 所有中间产物都是磷酸化的**

### 7 糖酵解分步机制（10步）

总反应与产能

- 投资阶段（1–5步）：消耗2 ATP，将己糖活化并裂解为2个C3。
- 收获阶段（6–10步）：每个C3产1 NADH + 2 ATP；总体每葡萄糖净得 2 ATP + 2 NADH + 2 丙酮酸。
- 总反应（细胞质）：Glucose + 2 ADP + 2 Pi + 2 NAD+ → 2 Pyruvate + 2 ATP + 2 NADH + 2 H+ + 2 H2O

Stage 1 准备/己糖阶段

1) 己糖激酶 Hexokinase（±Mg2+）

   - 反应：Glucose → Glucose-6-P（消耗1 ATP）
   - 机制：诱导契合，排斥水（变为非极性环境）；Mg2+ 屏蔽ATP负电，利于C6-OH进攻γ-P。
   - 组织同工酶（结构不同反应功能相同）：Hexokinase I–III（低Km，受G6P抑制）；肝脏/胰岛：Glucokinase（IV，Km高，不受G6P抑制，诱导型
   - 意义：将葡萄糖“捕获”在细胞内（带负电难外流）。

   ![1758681168475](image/notes/1758681168475.png)
2) 磷酸己糖异构酶 Phosphoglucose isomerase

   - 反应：G6P（醛糖）↔ F6P（酮糖），经烯二醇中间体。

   ![1758681806731](image/notes/1758681806731.png)
3) 磷酸果糖激酶-1 PFK-1（±Mg2+）

   - 反应：F6P → F1,6BP（消耗1 ATP），**强放能，不可逆（调节差->多途径->往往具有同工酶），限速/节律点**。
   - 调控：**ATP别构抑制**，AMP/ADP激活，柠檬酸抑制，F2,6BP强力激活（肝/心关键）。

   ![1758681940801](image/notes/1758681940801.png)
4) 醛缩酶 Aldolase

   - 反应：F1,6BP → GAP + DHAP（醇醛裂解）。

   ![1758682339939](image/notes/1758682339939.png)
5) 丙糖磷酸异构酶 Triose phosphate isomerase（TPI）

   - 反应：DHAP ↔ GAP（经烯二醇中间体）。
   - 生化：平衡偏向DHAP（96%），但后续消耗GAP推动正向；TPI缺陷罕见且致死。
   - 酶：八个beta sheet 桶装结构

   ![1758682403561](image/notes/1758682403561.png)

Stage 2 收获/三糖阶段
6) 甘油醛-3-P 脱氢酶 GAPDH

- 反应：GAP + Pi + NAD+ → 1,3-BPG + NADH + H+
- 耦联：**醛氧化（放能）与酰基磷酸形成（吸能）**通过硫酯中间体耦联（Cys活性位）。高磷酸转移电位

![1758682601587](image/notes/1758682601587.png)

7) 磷酸甘油酸激酶 PGK（±Mg2+）

   - 反应：1,3-BPG + ADP → 3-PG + ATP（底物水平磷酸化，第一分ATP）。
   - 与第6步通量耦联，整体前向。**为什么要有第七步？**
     因为第六步只是把能量**储存**成了一种“高能化学键”的形式（在1,3-BPG中），而第七步才是真正把这种能量**转化**成细胞可以直接使用的ATP的步骤。没有第七步，第六步储存的能量就无法被利用，整个糖酵解的能量产出目标就无法实现。

   ![1758682921380](image/notes/1758682921380.png)
8) 磷酸甘油酸变位酶 PGM

   - 反应：3-PG ↔ 2-PG（酶His-P参与；“此磷非彼磷”）。

   ![1758683135099](image/notes/1758683135099.png)
9) 烯醇化酶 Enolase

   - 反应：2-PG → PEP + H2O（脱水生成高能中间体）。

   ![1758683312951](image/notes/1758683312951.png)
10) 丙酮酸激酶 Pyruvate kinase（±K+，±Mg2+）

- 反应：PEP + ADP → Pyruvate + ATP（**第二个底物水平磷酸化**）。
- 驱动力：PEP转移后产物烯醇式→酮式互变异构，强放能。
- 调控：F1,6BP前馈激活；ATP、丙氨酸抑制；肝脏受磷酸化/去磷酸化调控（胰高血糖素抑制）。![1758683345369](image/notes/1758683345369.png)

补充：癌代谢与PKM2

- 正常：PKM1高活四聚体；肿瘤：PKM2占主导，易为低活形式，利于代谢物堆积供生物合成（Warburg效应）。
- 小分子激活PKM2促成四聚体化，是潜在治疗策略。

### 8 糖酵解的调控

层级与要点

- 膜转运：GLUT表达与转位（胰岛素↑GLUT4）；肝门静脉供糖波动由GLUT2缓冲。
- 酶学控制关键点（不可逆步）：HK/GK、PFK-1、PK。
- 细胞能荷：ATP/ADP/AMP；AMPK信号。
- 代谢物别构：柠檬酸（抑PFK-1）、F2,6BP（激PFK-1）、F1,6BP（激PK）。
- 激素与共价修饰（肝脏）
  - 胰岛素：促糖酵解与合成代谢（↑PFK-2活性→↑F2,6BP→激活PFK-1；去磷酸化激活PK）。
  - 胰高血糖素/肾上腺素：抑糖酵解促糖异生（PK磷酸化失活；PFK-2/FBPase-2切换至FBPase-2）。
- 组织特异
  - 肌肉：满足即时能量需求（运动时AMP↑，PFK-1↑）；休息时ATP/柠檬酸抑制。
  - 肝脏：维持血糖稳态，优先调控GK、PFK-1、PK及F2,6BP水平。

### 9 代谢旁路：磷酸戊糖途径（PPP）

两相反应

- 氧化相（不可逆）：G6P → 6-磷酸葡糖酸内酯 → 6-磷酸葡糖酸 → Ribulose-5-P + 2 NADPH + CO2
  - 限速酶：G6PDH（葡萄糖-6-磷酸脱氢酶）；RBC中为唯一生成NADPH的反应。
- 非氧化相（可逆，碳骨架重排）：R5P、Xu5P、S7P、E4P 与 F6P、GAP 互转
  - 转酮酶（TPP依赖）、转醛酶实现C2/C3转移，连接糖酵解。

生理与临床

- NADPH：脂肪酸/胆固醇合成、谷胱甘肽还原抗氧化、细胞色素P450。
- R5P：核苷酸/核酸合成。
- G6PDH缺乏：溶血性贫血（氧化应激触发：蚕豆、磺胺药、萘等）；全球~4亿人受影响。
- 区分NADH vs NADPH：前者偏分解代谢产能，后者偏合成代谢与抗氧化。

### 10 丙酮酸的去向（Fates of Pyruvate）

无氧条件（发酵）

- 乳酸发酵：Pyruvate + NADH ↔ Lactate + NAD+（乳酸脱氢酶，RBC/骨骼肌缺氧时）。
- 乙醇发酵（酵母/部分微生物）：
  - 丙酮酸脱羧酶（PDC，TPP依赖）→ 乙醛；乙醇脱氢酶（ADH）→ 乙醇（再生NAD+）。
  - 临床关联：TPP缺乏致脚气病/韦尼克-柯萨可夫综合征；ALDH缺乏（乙醛脱氢酶）致饮酒脸红；胎儿酒精综合征与乙醛毒性相关。
- 工业发酵：大规模生产酒精、有机酸、氨基酸等。

有氧条件（线粒体氧化）

- 丙酮酸脱氢酶复合体（PDH，TPP、FAD、NAD+、CoA、lipoate）：
  - Pyruvate → Acetyl-CoA + CO2 + NADH，进入柠檬酸循环与氧化磷酸化。
- 氧化产能差异（真核）：细胞质NADH入线粒体通过穿梭（苹果酸-天冬氨酸穿梭 2.5 ATP/NADH；甘油-3-磷酸穿梭 ~1.5 ATP/NADH）。

### 11 其他单糖进入糖酵解（Feeder pathways）

- 果糖：肝脏果糖激酶 → F1P → 醛缩酶B → DHAP/Glyceraldehyde → GAP；肌肉由HK直接→F6P。
- 半乳糖：Leloir途径（Gal → Gal-1P → UDP-Gal ↔ UDP-Glc → Glc-1P → Glc-6P）。
- 甘露糖：Man → Man-6P（HK）→ Fru-6P（磷酸甘露糖异构酶）。
- 应用：糖果工业用转化酶（invertase）水解蔗糖为葡萄糖+果糖，提升甜度与抗结晶。

### 12 Moonlighting（兼职）功能举例

- 糖酵解酶除催化外具调控/结构/信号功能。
- PKM2二聚体可入核调控转录因子（HIF-1α、STAT3），促进肿瘤相关基因表达；去磷酸化后四聚体化提高催化活性。

### 13 总结与要点回顾

- 糖酵解在胞质进行，将葡萄糖氧化为2丙酮酸，净得2 ATP与2 NADH；10个中间体均为磷酸化C3/C6化合物。
- 不可逆调控步：HK/GK、PFK-1、PK；PFK-1受F2,6BP强力激活是肝/心调控核心。
- 无氧条件下，丙酮酸在酵母→乙醇+CO2，在动物组织→乳酸；两者均再生NAD+维持糖酵解。
- PPP提供NADPH与R5P；G6PDH缺乏致溶血，临床常见。
- NADH偏产能，NADPH偏合成/抗氧化；细胞内池比例决定代谢方向。
- 代谢受多层调控：转运、别构、共价修饰、能荷、激素与组织特异性。

参考思考

- 为什么细胞把中间体全部磷酸化？（保留于胞内、能量暂存、提高专一性/降低活化能）
- 何以PEP具有极高转移电位？（烯醇→酮式互变提供额外驱动力）
- 肿瘤细胞为何偏好有氧糖酵解（Warburg效应）？（生物合成前体与NADPH需求）
