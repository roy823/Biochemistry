### 1 代谢导论回顾

- 定义
  - Catabolism 分解代谢：营养物/细胞组分降解，回收、供能。
  - Anabolism 合成代谢：由简单组分生物合成复杂分子。
- 三阶段框架
  - 阶段1：大分子↔单体（多糖/脂质↔单糖/脂肪酸/甘油等）
  - 阶段2：单体↔更简单有机物
  - 阶段3：与无机物互转（CO2/H2O/NH3）
- 代谢途径特征
  - 连续性（产物即下一步底物）；多步离散；可调控；可进化。
- 区室化（Compartmentation）
  - 真核细胞代谢在特定细胞器中进行，需跨膜转运。
  - 意义：隔离竞争反应、优化微环境、提高局部浓度、储存产物、实现级联。
  - 应用：人工细胞器/蛋白质胶囊/液-液相分离/细菌微室等，促进合成、隔离旁路、提高产率。
- 课前小测：为什么葡萄糖完全氧化分解为ATP有32 与30分子ATP的区别（两种电子传递方式消耗的ATP不同）

### 2 热力学与氧化还原基础

- 吉布斯自由能
  - ΔG = ΔH − TΔS；ΔG = ΔG′° + RT ln(Q)
  - ΔG < 0 反应自发向正向；ΔG = 0 平衡；ΔG > 0 倾向逆向。
  - 标准生化条件：pH 7，25°C，1 atm，活度~1 M。
  - 关联：ΔG′° = −RT ln Keq
- “高能化合物”
  - 定义：水解ΔG′° < −25 kJ/mol（数值更负）。
  - 代表：PEP（磷酸烯醇式丙酮酸）、1,3-BPG（1,3-二磷酸甘油酸）、磷酸肌酸等。
  - 磷酸基转移电位排序驱动磷酸基自高能向低能流动（激酶催化）。
- 氧化还原与能量
  - Nernst 方程；还原电位 E′° 越正，受电子能力越强。
  - 关系式：ΔG′° = −nFΔE′°（n为转移电子数，F为法拉第常数）。

### 3 电子载体与还原当量

- 电子转移方式
  - 直接电子转移；氢原子转移（H·）；氢负离子（:H−）；与氧气结合。
  - **还原当量**（reducing equivalent）：常以NADH、FADH2、QH2或H−计。
- 通用载体
  - NAD+/NADH、NADP+/NADPH、FAD/FADH2。
  - NAD+ 接受氢负离子（:H−）还原为 NADH；烟酰胺环可逆还原。
- 代谢分工与池浓度
  - [NAD+]/[NADH] 高，利于氧化（分解代谢）；[NADPH]/[NADP+] 高，利于还原（合成代谢）。
  - NAD+ 多参与氧化/产能；NADPH 供还原力（脂肪酸/胆固醇合成、抗氧化等）。
- 氧化还原酶、脱氢酶

### 5 糖的种类、消化吸收与细胞摄取

- 分类
  - 单糖：葡萄糖、果糖、半乳糖、核糖、2-脱氧核糖等；醛糖/酮糖；D-对映体为主。
  - 二糖：麦芽糖、蔗糖、乳糖；多糖：淀粉、糖原、纤维素。
- 消化与酶
  - 淀粉/糖原：α-淀粉酶切α-1,4键；麦芽糖/三糖：麦芽糖酶、α-葡萄糖苷酶；
    蔗糖：蔗糖酶（invertase）；乳糖：乳糖酶。
- 小肠吸收产物
  - 主要为葡萄糖/果糖/半乳糖，经门静脉入肝。
- 细胞摄取（GLUT家族）
  - GLUT1：广泛；GLUT2：肝/胰β细胞/肠；GLUT3：神经元；GLUT4：骨骼肌/心肌/脂肪，受胰岛素刺激转位；
    GLUT5：果糖转运。
  - 胰岛素激活受体酪氨酸激酶→GLUT4膜转位，促进摄糖。

### 6 糖酵解（Glycolysis）总览与发现史

- 定义与特征
- - 一分子葡萄糖转化为两分子丙酮酸，分为有氧和无氧。
  - 无需O2，胞质内进行；10步酶促，所有中间体均为磷酸化化合物。
  - 中心代谢通路，**进化上高度保守**；第一条被阐明的代谢途径之一。
- 发现史要点
  - Pasteur：发酵与酵母细胞相关；
  - Buchner：无细胞提取物可发酵（生化学诞生）；
  - Harden & Young：无细胞中的反应可以通过无机磷促进；中间体为磷酸化糖 果糖1,6-二磷酸 (Fructose 1,6-bisphosphate)；
  - Embden–Meyerhof–Parnas（EMP）途径建立。
- 葡萄糖的四大命运
  - 细胞外多糖（ECM/细胞壁）、储存（糖原/淀粉/蔗糖）、糖酵解供ATP/中间体、磷酸戊糖途径产R5P与NADPH。
- 重要生理
  - 脑（非饥饿）与红细胞主要依赖葡萄糖；RBC完全依赖糖酵解供能。
- 总览
  **十步反应 六种反应类型 所有的酶来自细胞质 所有中间产物都是磷酸化的**

### 7 糖酵解分步机制（10步）

总反应与产能

- 投资阶段（1–5步）：消耗2 ATP，将己糖活化并裂解为2个C3。
- 收获阶段（6–10步）：每个C3产1 NADH + 2 ATP；总体每葡萄糖净得 2 ATP + 2 NADH + 2 丙酮酸。
- 总反应（细胞质）：Glucose + 2 ADP + 2 Pi + 2 NAD+ → 2 Pyruvate + 2 ATP + 2 NADH + 2 H+ + 2 H2O

Stage 1 准备/己糖阶段

1) 己糖激酶 Hexokinase（±Mg2+）

   - 反应：Glucose → Glucose-6-P（消耗1 ATP）
   - 机制：诱导契合，排斥水（变为非极性环境）；Mg2+ 屏蔽ATP负电，利于C6-OH进攻γ-P。
   - 组织同工酶（结构不同反应功能相同）：Hexokinase I–III（低Km，受G6P抑制）；肝脏/胰岛：Glucokinase（IV，Km高，不受G6P抑制，诱导型
   - 意义：将葡萄糖“捕获”在细胞内（带负电难外流）。

   ![1758681168475](image/notes/1758681168475.png)
2) 磷酸己糖异构酶 Phosphoglucose isomerase

   - 反应：G6P（醛糖）↔ F6P（酮糖），经烯二醇中间体。

   ![1758681806731](image/notes/1758681806731.png)
3) 磷酸果糖激酶-1 PFK-1（±Mg2+）

   - 反应：F6P → F1,6BP（消耗1 ATP），**强放能，不可逆（调节差->多途径->往往具有同工酶），限速/节律点**。
   - 调控：**ATP别构抑制**，AMP/ADP激活，柠檬酸抑制，F2,6BP强力激活（肝/心关键）。

   ![1758681940801](image/notes/1758681940801.png)
4) 醛缩酶 Aldolase

   - 反应：F1,6BP → GAP + DHAP（醇醛裂解）。

   ![1758682339939](image/notes/1758682339939.png)
5) 丙糖磷酸异构酶 Triose phosphate isomerase（TPI）

   - 反应：DHAP ↔ GAP（经烯二醇中间体）。
   - 生化：平衡偏向DHAP（96%），但后续消耗GAP推动正向；TPI缺陷罕见且致死。
   - 酶：八个beta sheet 桶装结构

   ![1758682403561](image/notes/1758682403561.png)

Stage 2 收获/三糖阶段
6) 甘油醛-3-P 脱氢酶 GAPDH

- 反应：GAP + Pi + NAD+ → 1,3-BPG + NADH + H+
- 耦联：**醛氧化（放能）与酰基磷酸形成（吸能）**通过硫酯中间体耦联（Cys活性位）。高磷酸转移电位

![1758682601587](image/notes/1758682601587.png)

7) 磷酸甘油酸激酶 PGK（±Mg2+）

   - 反应：1,3-BPG + ADP → 3-PG + ATP（底物水平磷酸化，第一分ATP）。
   - 与第6步通量耦联，整体前向。**为什么要有第七步？**
     因为第六步只是把能量**储存**成了一种“高能化学键”的形式（在1,3-BPG中），而第七步才是真正把这种能量**转化**成细胞可以直接使用的ATP的步骤。没有第七步，第六步储存的能量就无法被利用，整个糖酵解的能量产出目标就无法实现。

   ![1758682921380](image/notes/1758682921380.png)
8) 磷酸甘油酸变位酶 PGM

   - 反应：3-PG ↔ 2-PG（酶His-P参与；“此磷非彼磷”）。

   ![1758683135099](image/notes/1758683135099.png)
9) 烯醇化酶 Enolase

   - 反应：2-PG → PEP + H2O（脱水生成高能中间体）。

   ![1758683312951](image/notes/1758683312951.png)
10) 丙酮酸激酶 Pyruvate kinase（±K+，±Mg2+）

- 反应：PEP + ADP → Pyruvate + ATP（**第二个底物水平磷酸化**）。
- 驱动力：PEP转移后产物烯醇式→酮式互变异构，强放能。
- 调控：F1,6BP前馈激活；ATP、丙氨酸抑制；肝脏受磷酸化/去磷酸化调控（胰高血糖素抑制）。![1758683345369](image/notes/1758683345369.png)

补充：癌代谢与PKM2

- 正常：PKM1高活四聚体；肿瘤：PKM2占主导，易为低活形式，利于代谢物堆积供生物合成（Warburg效应）。
- 小分子激活PKM2促成四聚体化，是潜在治疗策略。

### 8 糖酵解的调控

层级与要点

- 膜转运：GLUT表达与转位（胰岛素↑GLUT4）；肝门静脉供糖波动由GLUT2缓冲。
- 酶学控制关键点（不可逆步）：HK/GK、PFK-1、PK。
- 细胞能荷：ATP/ADP/AMP；AMPK信号。
- 代谢物别构：柠檬酸（抑PFK-1）、F2,6BP（激PFK-1）、F1,6BP（激PK）。

  ### 核心思想

  首先，我们要明白一个核心问题：**细胞为什么需要调节糖酵解？**

  因为糖酵解是细胞获取能量的核心途径，但能量（ATP）的生产需要根据细胞的“贫富”状态来精确控制。


  * 当细胞能量**充足**时（ATP多），就应该**减慢**糖酵解，节省葡萄糖资源。
  * 当细胞能量**匮乏**时（ATP少），就应该**加速**糖酵解，快速产生ATP。

  这种调控主要发生在糖酵解途径中**三个不可逆的、释放能量较多的步骤**上。这三个步骤的催化酶就是图中标注的三个主角。

  ---

  ### 1. 己糖激酶 (Hexokinase) - 糖酵解的“入口”

  这是糖酵解的第一步，负责将进入细胞的葡萄糖“捕获”。

  * **调控方式** : **产物抑制 (Product Inhibition)**
  * **抑制物** : **6-磷酸葡萄糖 (Glucose 6-phosphate)**
  * **生物学逻辑** : 己糖激酶的产物是6-磷酸葡萄糖 (G6P)。如果G6P在细胞内开始 **积累** ，通常意味着下游的反应（比如第三步的PFK-1）变慢了，细胞暂时处理不了这么多的葡萄糖。这时，积累的G6P就会反过来抑制己糖激酶的活性，告诉它：“下游堵车了，先别放新的葡萄糖进来了！” 这是一种非常直接有效的负反馈调节。

  ### 2. 磷酸果糖激酶-1 (Phosphofructokinase-1, PFK-1) - 糖酵解的“总开关”

  这是糖酵解的第三步，也是整个途径中 **最重要、最核心的调控点** ，被称为“限速步骤”。一旦分子通过了这一步，它基本上就注定要完成整个糖酵解过程。

  * **调控方式** : **别构调节 (Allosteric Regulation)**
  * **抑制物** :
  * **ATP** : ATP是糖酵解的最终能量产物。当细胞内**ATP水平很高**时，说明能量供应充足。ATP会作为别构抑制剂结合到PFK-1上，降低其活性。逻辑是：“我们已经很有钱了，没必要再拼命赚钱了。”
  * **柠檬酸 (Citrate)** : 柠檬酸是下游**三羧酸循环 (Krebs Cycle)** 的第一个中间产物。如果柠檬酸积累，说明下游代谢途径也很饱和，有充足的能量和生物合成原料。柠檬酸会抑制PFK-1，相当于告诉糖酵解：“下游的生产线也满了，源头请减速。”
  * **激活物** :
  * **AMP** : AMP是ATP消耗后的产物，是细胞 **能量匮乏的强烈信号** 。当AMP水平升高时，它会作为别构激活剂，强力激活PFK-1的活性。逻辑是：“我们快没钱了，赶紧开足马力赚钱！”
  * **2,6-二磷酸果糖 (Fructose 2,6-bisphosphate)** : 这是 **最强的PFK-1激活剂** 。它本身不是糖酵解的中间产物，而是一个专门的 **信号分子** 。它的浓度受到胰岛素等激素的调控，可以将身体的整体血糖状况与细胞内的糖酵解速率联系起来。

  ### 3. 丙酮酸激酶 (Pyruvate Kinase) - 糖酵解的“出口”

  这是糖酵解的最后一步，负责产生丙酮酸和ATP。

  * **调控方式** : **别构调节**
  * **抑制物** :
  * **ATP** : 和PFK-1的逻辑一样，高浓度的ATP标志着能量充足，因此需要抑制这个最终产出ATP的步骤。
  * **激活物** :
  * **1,6-二磷酸果糖 (Fructose 1,6-bisphosphate, FBP)** : 这是上游PFK-1反应的 **产物** 。这种调节方式非常巧妙，叫做 **前馈激活 (Feed-forward activation)** 。当上游的总开关PFK-1被激活并大量生产FBP时，FBP会“提前通知”下游的终点站丙酮酸激酶：“注意，一大波产物正在路上，请做好准备，提高你的工作效率！” 这保证了整个代谢途径的流动顺畅，避免了中间产物的拥堵。

  ### 总结

  这张PPT告诉我们，糖酵解的调控是一个精密的网络，主要通过以下方式实现：

  1. **能量状态感知** : 细胞通过ATP和AMP的相对水平来感知自身的能量“贫富”，从而调节关键酶的活性。
  2. **上下游信息互通** : 通过产物抑制（如HK）、下游产物反馈抑制（如柠檬酸）和上游产物前馈激活（如FBP），确保整个代谢流的协调与顺畅。
  3. **与整体代谢关联** : 通过柠檬酸等分子，将糖酵解的速率与其他代谢途径（如三羧酸循环）联系起来。

  ![1758685837711](image/notes/1758685837711.png)
- 激素与共价修饰（肝脏）

  - 胰岛素：促糖酵解与合成代谢（↑PFK-2活性→↑F2,6BP→激活PFK-1；去磷酸化激活PK）。
  - 胰高血糖素/肾上腺素：抑糖酵解促糖异生（PK磷酸化失活；PFK-2/FBPase-2切换至FBPase-2）。
- 组织特异

  - 肌肉：满足即时能量需求（运动时AMP↑，PFK-1↑）；休息时ATP/柠檬酸抑制。
  - 肝脏：维持血糖稳态，优先调控GK、PFK-1、PK及F2,6BP水平。

|                                 | ⑨   | 2-PG → 磷酸烯醇式丙酮酸 (PEP)      | ENO / 烯醇化酶              | 脱水；形成能量极高的烯醇式磷酸键。                  |
|                                 | ⑩   | PEP → 丙酮酸 (Pyruvate)            | PK / 丙酮酸激酶             | 第二次收获ATP；不可逆；糖酵解的终点。               |

### 9 代谢旁路：磷酸戊糖途径（PPP）

![1758686384444](image/notes/1758686384444.png)

两相反应

- 氧化相（不可逆）：G6P → 6-磷酸葡糖酸内酯 → 6-磷酸葡糖酸 → Ribulose-5-P + 2 NADPH + CO2
  - 限速酶：G6PDH（葡萄糖-6-磷酸脱氢酶）；RBC中为唯一生成NADPH的反应。
- 非氧化相（可逆，碳骨架重排）：R5P、Xu5P、S7P、E4P 与 F6P、GAP 互转
  - 转酮酶（TPP依赖）、转醛酶实现C2/C3转移，连接糖酵解。

好的，这张PPT展示的是生物体内一个非常重要的代谢途径—— **磷酸戊糖途径 (Pentose Phosphate Pathway, PPP)** 。

这个途径与我们熟知的糖酵解并行存在，都发生在**细胞质**中。但它的主要目的 **不是为了产生ATP** ，而是为了实现两个核心功能。这张图清晰地将该途径分为了两个阶段，下面我将为您详细讲解。

---

#### 磷酸戊糖途径 (PPP) 的核心目的

在深入细节之前，请记住这个途径的 **两大战略目标** ：

1. **生成 NADPH (还原型烟酰胺腺嘌呤二核苷酸磷酸)** ：这是一种高能电子载体，是细胞内主要的**还原力**来源。它对于 **合成代谢** （如脂肪酸、胆固醇的合成）和 **抗氧化应激** （保护细胞免受活性氧损伤）至关重要。
2. **生成 核糖-5-磷酸 (Ribose-5-phosphate)** ：这是合成**核苷酸**的前体物质。而核苷酸是构成 **DNA、RNA、ATP、NADH** 等核心分子的基石。

---

#### 第一阶段：氧化阶段 (Oxidative stage) - 不可逆

这个阶段是**产生NADPH**的主要部分，是不可逆的。

* **起始物** :  **葡糖-6-磷酸 (Glucose-6-phosphate)** ，它也是糖酵解的中间产物。
* **第一步** : 这是整个途径的 **关键调控步骤** 。在**葡糖-6-磷酸脱氢酶 (G6PD)** 的催化下，葡糖-6-磷酸被氧化，同时将 NADP⁺ 还原为  **NADPH** 。这是本阶段产生的 **第一个NADPH** 。
* **第二步和第三步** : 经过水解和第二次氧化脱羧反应（由**6-磷酸葡萄糖酸脱氢酶**催化），再次将一个 NADP⁺ 还原为  **NADPH** ，并释放一分子  **CO₂** 。这是本阶段产生的 **第二个NADPH** 。
* **此阶段终点** :  **核酮糖-5-磷酸 (Ribulose-5-phosphate)** ，这是一个五碳糖。

**氧化阶段总结：**
每消耗一个六碳的葡糖-6-磷酸，细胞就获得了 **2个NADPH** 和一个五碳的核酮糖-5-磷酸。

---

#### 第二阶段：非氧化阶段 (Non-oxidative stage) - 可逆

这个阶段是一系列复杂的、可逆的**糖分子重组（碳骨架重排）**过程。它的主要任务是根据细胞的需求，灵活地转化五碳糖。

* **起始物** :  **核酮糖-5-磷酸 (Ribulose-5-phosphate)** 。
* **五碳糖的转化** :
* 通过 **异构酶 (Isomerase)** ，它可以转变为 **核糖-5-磷酸 (Ribose-5-phosphate)** ，直接用于合成DNA和RNA。
* 通过 **差向异构酶 (Epimerase)** ，它可以转变为 **木酮糖-5-磷酸 (Xylulose-5-phosphate)** ，用于后续的重组反应。
* **碳骨架重排** : 接下来，在**转酮酶 (Transketolase)** 和**转醛酶 (Transaldolase)** 的作用下，不同碳数的糖分子开始进行碳单元的“剪切和粘贴”，例如：
* C5 + C5 → C3 + C7
* C3 + C7 → C4 + C6
* C4 + C5 → C3 + C6
* **此阶段终点产物** : 经过一系列重排，最终可以生成多种产物，其中最重要的是**果糖-6-磷酸 (Fructose-6-phosphate)** 和  **甘油醛-3-磷酸 (Glyceraldehyde-3-phosphate)** 。

#### 总结与意义

这张图最重要的信息是展示了磷酸戊糖途径的**灵活性**和它与 **糖酵解途径的联系** 。

1. **与糖酵解的联系** ：非氧化阶段的终点产物（果糖-6-磷酸和甘油醛-3-磷酸） **本身就是糖酵解的中间产物** 。这意味着磷酸戊糖途径和糖酵解是紧密相连、可以相互转化的。
2. **途径的灵活性** ：细胞可以根据自身需求，灵活地调控这条途径的走向：

* **当细胞需要大量分裂，急需DNA/RNA时** : 细胞会优先将五碳糖转化为 **核糖-5-磷酸** 。
* **当细胞需要大量合成脂肪酸，急需NADPH时** : 细胞会完整地走完两个阶段。非氧化阶段生成的果糖-6-磷酸和甘油醛-3-磷酸可以重新变回葡糖-6-磷酸，再次进入氧化阶段，循环往复地 **最大化生产NADPH** 。
* **当细胞能量不足时** : 产生的果糖-6-磷酸和甘油醛-3-磷酸可以直接进入糖酵解途径，继续分解以 **产生ATP** 。

因此，磷酸戊糖途径是细胞代谢网络中一个至关重要的“枢纽”，它根据细胞是需要“建筑材料”（核糖）还是需要“还原电力”（NADPH），来智能地调配葡萄糖的去向。

生理与临床

- NADPH：脂肪酸/胆固醇合成、谷胱甘肽还原抗氧化、细胞色素P450。
- R5P：核苷酸/核酸合成。
- G6PDH缺乏：溶血性贫血（氧化应激触发：蚕豆、磺胺药、萘等）；全球~4亿人受影响。
- 区分NADH vs NADPH：前者偏分解代谢产能，后者偏合成代谢与抗氧化。

### 10 丙酮酸的去向（Fates of Pyruvate）

无氧条件（发酵）

- 乳酸发酵：Pyruvate + NADH ↔ Lactate + NAD+（乳酸脱氢酶，RBC/骨骼肌缺氧时）。
- 乙醇发酵（酵母/部分微生物）：
  - 丙酮酸脱羧酶（PDC，TPP依赖）→ 乙醛；乙醇脱氢酶（ADH）→ 乙醇（再生NAD+）。
  - 临床关联：TPP缺乏致脚气病/韦尼克-柯萨可夫综合征；ALDH缺乏（乙醛脱氢酶）致饮酒脸红；胎儿酒精综合征与乙醛毒性相关。
- 工业发酵：大规模生产酒精、有机酸、氨基酸等。

有氧条件（线粒体氧化）

- 丙酮酸脱氢酶复合体（PDH，TPP、FAD、NAD+、CoA、lipoate）：
  - Pyruvate → Acetyl-CoA + CO2 + NADH，进入柠檬酸循环与氧化磷酸化。
- 氧化产能差异（真核）：细胞质NADH入线粒体通过穿梭（苹果酸-天冬氨酸穿梭 2.5 ATP/NADH；甘油-3-磷酸穿梭 ~1.5 ATP/NADH）。

### **引言：从糖酵解到细胞呼吸**

糖酵解 (Glycolysis) 在细胞质中将一分子葡萄糖分解为两分子丙酮酸 (Pyruvate)，并产生少量的ATP和NADH。在有氧条件下，细胞呼吸 (Cellular Respiration) 会将丙酮酸彻底氧化为 **C**O**2** 和 **H**2O，释放大量能量。柠檬酸循环是细胞呼吸的核心，是连接三大营养物质（糖、脂肪、蛋白质）分解代谢的中心枢纽。

细胞呼吸通常分为三个主要阶段：

1. **乙酰辅酶A的生成** ：丙酮酸被氧化脱羧，生成乙酰辅酶A (Acetyl-CoA)。
2. **乙酰辅酶A的氧化** ：乙酰辅酶A进入柠檬酸循环，其乙酰基被彻底氧化为 **C**O**2**，能量以NADH和**F**A**D**H**2**的形式被捕获。
3. **电子传递与氧化磷酸化** ：NADH和**F**A**D**H**2**将高能电子传递给电子传递链，最终生成大量ATP。

---

### **1. 柠檬酸循环（TCA循环）概述**

* **定义** ：柠檬酸循环是一系列在**线粒体基质**中发生的酶促反应，它彻底氧化乙酰CoA中的乙酰基，产生 **C**O**2**、GTP/ATP以及大量的还原当量 (NADH, **F**A**D**H**2**)。
* **发生部位** ：
* **真核细胞** ：绝大多数反应发生在线粒体基质 (Mitochondrial Matrix) 中。
* **例外** ：**琥珀酸脱氢酶 (Succinate Dehydrogenase)** 是一个例外，它镶嵌在线粒体内膜上，同时也是电子传递链的复合物II。
* **核心功能** ：
* **分解代谢 (Catabolism)** ：作为所有产能营养素（糖类、脂肪酸、氨基酸）分解代谢的共同通路。
* **合成代谢 (Anabolism)** ：为生物合成（如氨基酸、血红素、葡萄糖）提供前体物质，是名副其实的“代谢十字路口”。
* **循环载体** ：草酰乙酸 (Oxaloacetate, OAA) 在循环的第一步与乙酰CoA缩合，并在最后一步被再生，理论上一分子OAA可以催化无数分子乙酰CoA的氧化。

---

### **2. 乙酰CoA的生成：连接糖酵解与TCA循环的桥梁**

#### **2.1 丙酮酸进入线粒体**

糖酵解产生的丙酮酸通过线粒体外膜的孔道，再由内膜上的**线粒体丙酮酸载体 (Mitochondrial Pyruvate Carrier, MPC)** 主动转运进入线粒体基质。

#### **2.2 丙酮酸脱氢酶复合体 (PDH Complex)**

在线粒体基质中，丙酮酸被一个巨大的多酶复合体——**丙酮酸脱氢酶复合体 (PDH)** 催化，发生 **氧化脱羧反应** ，生成乙酰CoA。 **此反应不可逆** ，是决定丙酮酸命运的关键一步。

![1760150652607](image/notes/1760150652607.png)

* **总反应** :
  **Pyruvate**+**CoA-SH**+**NAD**+**→**Acetyl-CoA**+**CO**2+**NADH**+**H**+**
* **复合体构成** :
* **E1 - 丙酮酸脱氢酶 (Pyruvate Dehydrogenase)** ：核心催化脱羧反应。
* **E2 - 二氢硫辛酰胺乙酰转移酶 (Dihydrolipoyl Transacetylase)** ：构成复合体的核心骨架，负责转移乙酰基。
* **E3 - 二氢硫辛酰胺脱氢酶 (Dihydrolipoyl Dehydrogenase)** ：负责再生氧化型的辅酶。
* **五种辅酶/辅基** :

1. **焦磷酸硫胺素 (TPP)** ：结合在E1上，负责脱羧。
2. **硫辛酸/硫辛酰胺 (Lipoic Acid/Lipoamide)** ：以酰胺键共价连接在E2的赖氨酸残基上，形成一个长的“摇臂”，负责传递乙酰基和电子。
3. **黄素腺嘌呤二核苷酸 (FAD)** ：紧密结合在E3上，接受电子。
4. **辅酶A (CoA)** ：作为乙酰基的载体。
5. **烟酰胺腺嘌呤二核苷酸 (**N**A**D+) ：最终的电子受体。

![1760150899377](image/notes/1760150899377.png)


 **催化步骤** ：

1. **脱羧 (Decarboxylation)** ：丙酮酸进入E1的活性位点，在其辅酶TPP（焦磷酸硫胺素）的帮助下，脱去一个羧基，释放出一分子 **C**O**2**。剩下的两碳片段形成“羟乙基-TPP”中间体。

   ![1760151190425](image/notes/1760151190425.png)
2. **氧化与转移 (Oxidation & Transfer)** ：羟乙基被从E1转移到E2的辅酶——硫辛酰胺（Lipoamide）的长臂上。在这个过程中，羟乙基被氧化成乙酰基，同时硫辛酰胺被还原成二氢硫辛酰胺。这个长臂就像一个起重机，在不同酶的活性位点间摆动。

   ![1760151241061](image/notes/1760151241061.png)
3. **乙酰辅酶A的形成 (Acetyl-CoA Formation)** ：E2催化，将乙酰基从二氢硫辛酰胺臂上转移给辅酶A (CoA-SH)，形成最终产物 **乙酰辅酶A (Acetyl-CoA)** 。乙酰辅酶A会离开复合体，进入柠檬酸循环。

   ![1760151263362](image/notes/1760151263362.png)
4. **E2辅酶的再生 (Reoxidation of Lipoamide)** ：携带两个氢原子的二氢硫辛酰胺臂必须被重新氧化，才能参与下一轮反应。它将氢和电子传递给E3酶上的辅基FAD，使其还原为**F**A**D**H**2**，自身则恢复为氧化态。

   ![1760151286615](image/notes/1760151286615.png)
5. **E3辅酶的再生与NADH的生成 (Regeneration of FAD and Formation of NADH)** ：最后，E3上的**F**A**D**H**2**将电子传递给**N**A**D**+，生成**NADH**和**H**+，同时FAD被重新氧化，使整个复合体恢复到初始状态，准备催化下一个丙酮酸分子。

   ![1760151319668](image/notes/1760151319668.png)

 **小结** ：PDH复合体通过这种高效的流水线作业，完成了丙酮酸的氧化脱羧，其产物——**乙酰CoA**和 **NADH** ——都是高能分子。

#### **2.3 乙酰CoA：高能硫酯化合物**

乙酰CoA的乙酰基通过一个**硫酯键 (Thioester Bond)** 与辅酶A连接。由于硫酯键缺乏共振稳定性，其水解的吉布斯自由能变 (**Δ**G**′**∘**≈**−**31.5** kJ/mol) 比ATP水解更为负，是一种高能化合物，为后续柠檬酸的合成提供了强大的化学驱动力。

#### **2.4 临床关联：砷中毒 (Arsenic Poisoning)**

砷化物（如亚砷酸盐）可以与含有**硫辛酰胺**的酶（如PDH复合体和α-酮戊二酸脱氢酶复合体）中的巯基发生高亲和力的结合，形成稳定的二配位加合物，从而不可逆地抑制这些酶的活性，阻断细胞呼吸和能量产生，导致严重中毒。

---

### **3. 柠檬酸循环八步反应机制**

1. **缩合反应 (Condensation)**
   * **反应** : 乙酰CoA + 草酰乙酸 → 柠檬酸 + CoA-SH
   * **酶** : **柠檬酸合酶 (Citrate Synthase)**
   * **要点** : 这是一个由高能硫酯键水解放能驱动的羟醛缩合反应， **高度放能且不可逆** 。酶采用“诱导契合”机制，先结合草酰乙酸，构象改变后再结合乙酰CoA，防止乙酰CoA被无效水解。
2. **异构化 (Isomerization)**
   * **反应** : 柠檬酸 ↔ 异柠檬酸
   * **酶** : **顺乌头酸酶 (Aconitase)**
   * **要点** : 通过脱水和再水合两步反应，将叔醇（柠檬酸）转化为更易于氧化的仲醇（异柠檬酸）。柠檬酸是一个对称分子，但酶通过“ **三点附着** ”的方式使其发生不对称催化。该酶含有一个铁硫簇，并被 **氟乙酸** （在体内转化为氟柠檬酸）强烈抑制。
3. **第一次氧化脱羧 (Oxidative Decarboxylation)**
   * **反应** : 异柠檬酸 + **N**A**D**+ → α-酮戊二酸 + **C**O**2** + NADH
   * **酶** : **异柠檬酸脱氢酶 (Isocitrate Dehydrogenase)**
   * **要点** : 循环中 **第一个产生NADH和**C**O**2的步骤，是关键的**速率调节步骤** 。
4. **第二次氧化脱羧 (Oxidative Decarboxylation)**
   * **反应** : α-酮戊二酸 + **N**A**D**+ + CoA-SH → 琥珀酰CoA + **C**O**2** + NADH
   * **酶** : **α-酮戊二酸脱氢酶复合体 (α-Ketoglutarate Dehydrogenase Complex)**
   * **要点** : 机制、结构和所需辅酶均与 **PDH复合体高度相似** ，也是一个关键的 **不可逆调控点** 。反应生成另一个高能硫酯化合物——琥珀酰CoA。
5. **底物水平磷酸化 (Substrate-Level Phosphorylation)**
   * **反应** : 琥珀酰CoA + GDP/ADP + Pi → 琥珀酸 + GTP/ATP + CoA-SH
   * **酶** : **琥珀酰CoA合成酶 (Succinyl-CoA Synthetase)**
   * **要点** : 利用琥珀酰CoA的高能硫酯键断裂释放的能量，直接合成GTP（主要在动物）或ATP（在植物和细菌中），这是TCA循环中**唯一一次直接产生高能磷酸键**的反应。
6. **脱氢 (Dehydrogenation)**
   * **反应** : 琥珀酸 + FAD → 延胡索酸 + **F**A**D**H**2**
   * **酶** : **琥珀酸脱氢酶 (Succinate Dehydrogenase)**
   * **要点** : 循环中 **唯一的膜结合酶** ，直接将电子传递给电子传递链中的辅酶Q (生成**Q**H**2**)。因此，教科书中有时也将产物写作**Q**H**2**而非**F**A**D**H**2**，后者是酶的辅基。
7. **水合 (Hydration)**
   * **反应** : 延胡索酸 + **H**2O → L-苹果酸
   * **酶** : **延胡索酸酶 (Fumarase)**
   * **要点** : 高度立体专一性，只作用于反式的延胡索酸生成L-苹果酸。
8. **脱氢 (Dehydrogenation)**
   * **反应** : L-苹果酸 + **N**A**D**+ → 草酰乙酸 + NADH
   * **酶** : **苹果酸脱氢酶 (Malate Dehydrogenase)**
   * **要点** : 此反应的标准自由能变为正，平衡偏向苹果酸。但在细胞内，由于草酰乙酸在第一步反应中被柠檬酸合酶不断消耗，其浓度极低，从而 **拉动反应持续向右进行** ，完成循环。

### **4. 能量收支与调控**

#### **4.1 能量收支**

每循环一轮（氧化一分子乙酰CoA）：

* **产出** :
* **2** 分子 **C**O**2**
* **3** 分子 NADH
* **1** 分子 **F**A**D**H**2**
* **1** 分子 GTP (或 ATP)
* **ATP当量** : 在氧化磷酸化中，每分子NADH约产生**2.5**个ATP，每分子**F**A**D**H**2**约产生**1.5**个ATP。因此，一轮TCA循环总共产生约 **10** 个ATP。
* **每分子葡萄糖** : 经糖酵解、PDH反应和两轮TCA循环，一分子葡萄糖完全氧化大约产生 **30-32** 个ATP。

#### **4.2 循环的调控**

TCA循环的速率主要受细胞的**能量状态**调控，调控点集中在三个**不可逆**的放能步骤：

1. **丙酮酸脱氢酶复合体 (PDH)**
2. **柠檬酸合酶 (Citrate Synthase)**
3. **异柠檬酸脱氢酶 (Isocitrate Dehydrogenase)**
4. **α-酮戊二酸脱氢酶复合体 (α-Ketoglutarate Dehydrogenase Complex)**

* **抑制信号 (能量充足)** ：
* 高水平的 **ATP**
* 高水平的 **NADH** (产物抑制)
* 下游产物反馈抑制 (如 **柠檬酸** 、 **琥珀酰CoA** )
* **激活信号 (能量需求)** ：
* 高水平的 **ADP** (能量低的信号)
* **C**a**2**+**** 离子 (如在肌肉收缩时，是需要能量的信号)

### **5. 旁路与补充反应**

#### **5.1 回补反应 (Anaplerotic Reactions)**

由于TCA循环的中间产物会被不断抽出用于生物合成，需要有“回补”反应来补充它们，以维持循环的正常运转。最重要的回补反应是：

* **丙酮酸羧化酶 (Pyruvate Carboxylase)** ：
  **Pyruvate**+**CO**2+**ATP**→**Oxaloacetate**+**ADP**+**Pi**

  该酶在肝脏和肾脏中活性很高，由乙酰CoA别构激活。

#### **5.2 乙醛酸循环 (Glyoxylate Cycle)**

* **存在** : 植物、真菌和某些微生物特有，不存在于脊椎动物。
* **目的** : 允许生物体将**脂肪 (乙酰CoA) 净合成为碳水化合物** (如葡萄糖)。
* **关键酶** :

1. **异柠檬酸裂解酶 (Isocitrate Lyase)**
2. **苹果酸合酶 (Malate Synthase)**

* **机制** : 绕过了TCA循环中两个释放**C**O**2**的脱羧步骤，使得两分子的乙酰CoA可以净生成一分子的琥珀酸，琥珀酸进而可以转化为草酰乙酸用于糖异生。

---

### **6. 总结与要点回顾**

1. TCA循环是 **有氧代谢的核心** ，在线粒体基质中将乙酰CoA彻底氧化为**C**O**2**。
2. **PDH复合体**是连接糖酵解和TCA循环的 **不可逆桥梁** ，其机制复杂，需要五种辅酶。
3. 每轮循环产生  **3 NADH、1 **F**A**D**H**2**、1 GTP 和 2 **C**O**2 ，为氧化磷酸化提供大量还原力。
4. 循环的速率受到细胞**能量状态**的严格调控，主要调控点是几个不可逆反应的酶。
5. TCA循环不仅是能量工厂，也是 **生物合成的原料库** ，需要**回补反应**来维持中间产物的稳定。
6. **乙醛酸循环**是某些生物将脂肪转化为糖的特有途径。
